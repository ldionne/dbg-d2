Instructions

Thanks for downloading dbg, a C++ library that provides x86/x64 stack frame enumeration, a collection of powerful assertions and other debugging features.

Please visit http://bitbucket.org/edd/dbg/wiki/Home for more information on dbg, including compilation and usage instructions.

— Edd

Version History

Version 0.2.1 released on 14 March 2013

  • Further improved symbol lookup speed for MinGW builds.
  • The use of fungo can now be disabled by defining DBG_DONT_USE_FUNGO during builds of the library and any client code. This is particularly relevant for C++11 projects, where std::exception_ptr can instead be used for opaque exception storage and propagation.
  • Fixed an issue that caused build failures on Windows when UNICODE was defined.

Version 0.2.0 released on 17 February 2013

  • Added DBG_IMPURE_UNUSUAL and DBG_HOT_IMPURE_UNUSUAL macros.
  • The symbol lookup implementation for MinGW now looks in the COFF symbol table, rather than in the DWARF data. This approach is faster and simpler.

Version 0.1.1 released on 11 September 2012

  • Environment variables are now restored in tests that modify them.

Version 0.1.0 released on 08 September 2012

  • Initial release.
